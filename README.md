# ![iwhale-logo](./images/common/iwhale-logo.png) iwhale-ui-weapp

#### 介绍

> 微信小程序UI组件及商城模板

#### 实现组件

##### 基础组件

* 轮播图（swiper）
* 自定义顶部导航栏（navbar）
* 菜单标签组（menu-tabs）
* 按钮（button）
* 输入框（input）
* 数字输入框（input-number）
* 标签页（tabs）
* 头像（avatar）
* 卡片（card）
* 无线信号（signal）

##### 业务组件
* 商品列表

##### 插件
* wx-validate（表单校验）
